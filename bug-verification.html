BUG-1：非链接元素跳转链接不生效 严重

问题：元素刚标记时 modifiedHref 为空，click 事件不注册； 用户在检查面板设置跳转链接后，点击元素不会跳转。
修复：始终注册 click 事件，handler 内部判断是否有链接才跳转；同时动态更新 cursor 样式。

📋 验证步骤：

  1. 点击扩展图标 → 显示工具栏，点击「选择元素」进入选择模式
  2. 点击下面的紫色方框，标记它
  3. 点击元素上的编号徽章，打开检查面板
  4. 在「跳转链接」输入框中填入：https://www.baidu.com
  5. 关闭检查面板，退出选择模式
  6. 点击紫色方框 → 观察是否跳转到百度（新标签页打开）
  7. 额外验证：鼠标悬停在方框上时，光标应该变成 👆 pointer 形状
✅ 修复后预期：点击紫色方框后，应在新标签页打开百度；鼠标悬停时光标变手型。

BUG-2：滚轮缩放用 width/height 而非 transform 中等

问题：滚轮缩放通过修改 width/height 实现，文字不缩放、破坏文档流布局。
修复：改用 transform: scale() 实现真正的视觉缩放，不影响布局。

📋 验证步骤：

  1. 标记下面的黄色方框
  2. 按住 Ctrl（Mac 为 Cmd）+ 滚轮向上放大
  3. 观察：方框内的文字是否跟着一起放大
  4. 观察：方框下方的其他元素位置是否被推动（应该不动才对）
  5. 再滚轮向下缩小，观察文字是否一起缩小
✅ 修复后预期：文字和方框整体缩放（像放大镜效果）；下面的绿色参考元素位置不变（不破坏文档流）。

BUG-3：导出 Diff 后页面原有事件全部丢失 严重

问题：buildDiffData 用 innerHTML = old 恢复元素，重建 DOM 导致所有 addEventListener 绑定的事件全部丢失。
修复：用 cloneNode(true) 克隆副本后操作，原始 DOM 完全不动。

📋 验证步骤：

  1. 先点击下面的「点我测试」蓝色按钮，观察下方显示「按钮点击了 1 次」，说明事件正常工作
  2. 标记这个蓝色按钮
  3. 点击工具栏的「导出 Diff」按钮，生成并下载 Diff 文件
  4. 导出完成后，再次点击「点我测试」按钮
  5. 观察下方计数器是否继续增加（说明事件还在）
✅ 修复后预期：导出 Diff 后再点击按钮，计数器应该继续计数（事件监听器仍然有效）。
❌ 未修复表现：导出后点击按钮无反应，计数器不增加。

BUG-4：三态切换时检查面板位置丢失 轻微

问题：toggleThreeState 直接 remove 检查面板，不保存位置。
修复：移除前先读取位置保存到 state.inspectorPos。

📋 验证步骤：

  1. 标记下面的绿色方框
  2. 点击编号徽章打开检查面板
  3. 拖拽检查面板的标题栏，把它拖到屏幕左侧（或你喜欢的位置）
  4. 按快捷键 Ctrl+Shift+E（Mac 为 Cmd+Shift+E）隐藏工具栏
  5. 再按一次 Ctrl+Shift+E 显示工具栏（唤醒按钮）
  6. 再按一次显示完整工具栏
  7. 再次点击编号徽章打开检查面板
  8. 观察检查面板的位置是否和你之前拖动的位置一致
✅ 修复后预期：三态切换后再次打开检查面板，位置保持在你上次拖动的地方。
❌ 未修复表现：面板位置回到默认的右上角。
