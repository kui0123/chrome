## Mark2AI - Chrome 扩展（v2.0.0）
### 核心定位
一款用于在浏览器中 标记 HTML 组件、可视化编辑 CSS 样式和 HTML 内容 ，并生成 diff 文件供 AI Agent 迭代使用的 Chrome MV3 扩展程序。

### 核心能力
能力领域 主要功能 组件标记 点击页面元素进行标记，支持多选和组合标记 可视化编辑 拖拽移动、调整大小、滚轮缩放、双击编辑文本 样式精修 CSS 属性编辑（位置/尺寸/颜色/边框/圆角/字体/背景） 结构操作 复制/删除/添加 DOM 元素，支持批量操作 主题系统 4 套预设主题（暮紫/深藏青/灰绿/暖棕）+ 自定义颜色 Diff 导出 一键生成 Markdown 和 JSON 格式的修改报告

### 交互方式
- 浮动工具栏 ：右上角三态切换（隐藏 → 唤醒按钮 → 完整工具栏）
- 检查面板 ：点击编号徽章打开，支持 10 个编辑分区的精确调整
- 快捷键 ： Alt+E （三态切换）、 Alt+"+" （快速选择模式）
- 右键菜单 ：快速触发选择组件、生成 Diff、清除标记
### 技术特点
- 样式隔离： all: initial/unset 防止宿主页面 CSS 冲突
- 状态持久化：标记数据存储于 sessionStorage ，刷新保留
- 对齐辅助线：自动检测尺寸和位置对齐关系
- 字体管理：支持 15 种预设字体 + 自定义字体，三态预览提示
### 项目结构
```
extension/                    # Chrome 扩展加载目录
├── manifest.json             # 扩展配置
├── background/background.js  # Service Worker
├── content/content.js        # 页面内核心脚本
├── content/content.css       # 页面内样式
└── icons/                    # 扩展图标
```
### 使用场景
主要面向 Vibe Coder、产品·设计·运营、无代码创作者 ，通过可视化操作快速调整页面元素，生成的 Diff 文件可直接用于指导 AI Agent 进行代码迭代。
