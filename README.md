# HTML Diff Marker

> 版本：1.0.0

在 Chrome 中标记 HTML 组件的修改，并生成 diff 文件供 AI Agent 迭代使用。

## 功能特性

- 🎯 **元素选择与标记**：点击选择页面元素，自动添加编号标记
- 🎨 **可视化编辑**：拖拽移动、8 方向调整大小、滚轮缩放、双击编辑文本
- 📐 **智能对齐**：位置和尺寸调整时自动显示对齐辅助线
- 📝 **编辑面板**：修改说明、链接地址、位置、大小、样式属性、HTML 代码
- 📋 **元素复制**：一键复制元素并插入到原位置后面，刷新后自动恢复
- 💾 **状态持久化**：所有标记和修改自动保存到 sessionStorage，刷新不丢失
- 📤 **Diff 导出**：一键导出 Markdown 和 JSON 格式的差异报告
- ⌨️ **快捷键支持**：Ctrl/Cmd + Shift + E 三态切换工具栏
- 🎛️ **可拖拽面板**：工具栏和编辑面板均可自由拖拽定位

## 安装步骤

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目根目录
5. 扩展加载成功后，工具栏会显示扩展图标

## 使用方法

### 基础操作流程

| 步骤 | 操作 | 说明 |
| ---- | ---- | ---- |
| 1 | 点击扩展图标 | 弹出控制面板，或显示唤醒按钮 |
| 2 | 点击「选择元素」 | 进入选择模式，鼠标变为十字线 |
| 3 | 点击页面元素 | 标记该元素，自动打开编辑面板 |
| 4 | 在编辑面板中修改 | 调整样式、位置、大小、文本等 |
| 5 | 点击「保存修改」 | 保存修改并关闭面板 |
| 6 | 点击「导出 Diff」 | 生成并下载差异报告文件 |

### 详细功能说明

#### 标记元素
- 点击工具栏的「选择元素」按钮进入选择模式
- 鼠标悬停时元素显示蓝色虚线边框
- 点击元素完成标记，显示绿色边框和编号徽章
- 标记后自动打开编辑面板

#### 编辑面板
点击元素顶部的编号徽章打开编辑面板，包含以下功能：

- **组件标签 / 备注**：自定义标记名称或修改说明
- **链接地址 (href)**：仅对 `<a>` 元素显示，可修改链接
- **位置调整**：X/Y 坐标微调，支持 ±1px / ±10px 按钮
- **大小调整**：宽度/高度微调，支持 ±1px / ±10px 按钮
- **样式编辑**：背景颜色、文本颜色、字体大小、字体粗细、内边距、外边距、圆角、边框、显示方式
- **HTML 编辑**：查看原始 HTML，输入修改后的 HTML 代码
- **复制按钮**：一键复制当前元素，插入到原元素后面
- **删除按钮**：删除当前标记
- **保存按钮**：保存所有修改

#### 可视化编辑

**拖拽移动**：直接拖动标记的元素改变位置

**调整大小**：拖动元素边缘和角落的紫色把手（共 8 个方向）

**滚轮缩放**：按住 Ctrl/Cmd + 滚轮，按比例缩放元素

**文本编辑**：双击元素进入可编辑模式，修改文本内容
- 按 Enter 保存并退出
- 按 Escape 取消并退出
- 点击元素外部保存并退出

**智能对齐**：拖动位置或调整大小时，5px 阈值内自动吸附对齐，并显示红色对齐辅助线

#### 元素复制
- 在编辑面板中点击「📋 复制」按钮
- 自动复制当前元素并插入到原元素后面
- 复制的元素会被自动标记
- **刷新页面后自动恢复**（持久化存储）

#### 工具栏三态切换
使用快捷键 **Ctrl/Cmd + Shift + E** 在三种状态间循环：
1. **隐藏**：不显示任何界面
2. **唤醒按钮**：显示一个小的悬浮按钮
3. **完整工具栏**：显示完整的工具栏面板

#### 清除所有标记
点击工具栏的「清空」按钮，确认后清除所有标记和修改。

## 导出文件格式

### Markdown 格式 (.md)
包含每个组件的详细差异信息：
- 元素信息和 CSS 选择器
- 修改说明
- 样式变更列表
- href 变更（如有）
- 原始 HTML 代码
- 修改后的 HTML 代码
- 行级 Diff 对比
- 完整 JSON 数据

### JSON 格式 (.json)
结构化数据，包含：
- `url`：页面地址
- `title`：页面标题
- `timestamp`：导出时间
- `totalMarked`：标记组件总数
- `totalModified`：有修改的组件数
- `items`：组件数组，每个包含原始/修改后的 HTML、样式变更、修改说明等

## 注意事项

- 本扩展仅用于前端页面设计和开发调试
- 所有修改仅在本地浏览器中生效，不会修改实际网站
- 标记数据存储在 sessionStorage 中，关闭标签页后会清除
- 对于复杂的 DOM 结构，选择器可能不够精确，刷新后可能无法正确恢复标记
- 复制的元素会在刷新后自动重建，但复杂的交互功能可能无法完全保留

## 代码规范（开发者必读）

本项目有严格的开发规范，开发前请务必阅读 [Project_Rule.md](./Project_Rule.md)。

核心原则：
- **Rule 0**：任何文件编辑前必须先 Read
- **信息重复原则**：核心源文件的代码/内容，永远不应该以字符串形式存在于项目中的任何其他文件里
- **禁止代码生成器**：禁止创建任何形式的代码生成器脚本（如 generate_ext.py）
- **直接编辑源文件**：直接 Edit 或 Write 目标文件，不经过中间脚本

## 故障排除

### 扩展无法加载
- 检查 manifest.json 格式是否正确
- 确保所有引用的文件都存在
- 查看扩展管理页面的错误信息

### 选择模式不生效
- 刷新页面后重试
- 检查是否有其他扩展冲突
- 查看控制台是否有 JavaScript 错误

### 刷新后标记消失
- 确认使用的是 sessionStorage（同一标签页内有效）
- 复杂页面中选择器可能不稳定，尝试标记更具体的元素
- 复制的元素会自动恢复，普通标记依赖选择器匹配

### 编辑面板打不开
- 确保点击的是元素顶部的编号徽章
- 如果徽章被遮挡，尝试调整元素位置

### 导出失败
- 检查浏览器下载权限
- 尝试禁用其他下载管理扩展
- 页面上下文丢失时会降级为直接下载 Markdown 文件
